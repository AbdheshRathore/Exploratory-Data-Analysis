Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on historical weather data to understand climate patterns, seasonal trends, and relationships between key weather variables. The analysis helps uncover insights that can support weather forecasting modeling.

Objectives

Understand historical weather patterns and trends
Analyze relationships between temperature, humidity, rainfall, and wind speed
Identify seasonal variations and extreme weather conditions
Prepare a clean dataset for future forecasting models
